You only need to do this once. Your data will then save automatically to Supabase on every change.
1
Go to supabase.com, sign up free with your personal Gmail and create a new project
2
In your project go to Settings → API and copy your Project URL and anon public key
3
Go to SQL Editor in Supabase and run this query:
CREATE TABLE tracker_data (user_id text PRIMARY KEY, data jsonb, updated_at timestamptz DEFAULT now());

Paste your keys below
